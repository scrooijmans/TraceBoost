# Documentation

This directory contains the initial documentation baseline for TraceBoost.

The current project direction is a Tauri desktop application for seismic data upscaling. The first upscaling mode will focus on learned super-resolution for rasterized seismic sections and slices.

The files in this directory should be treated as working material:

- architecture analysis
- product direction notes
- technical references
- imported baseline docs that will be replaced or rewritten over time
