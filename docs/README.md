# Documentation

This directory contains the current TraceBoost architecture docs plus archived legacy imports.

TraceBoost is also the repository where the relationship between the first-party app and the platform is explained:

- `TraceBoost` is the first-party workflow app
- `Ophiolite` is the subsurface platform core
- `Ophiolite Charts` is the visualization SDK
